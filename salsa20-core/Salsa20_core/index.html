Module Salsa20_core

The Salsa20 specification specifies a Salsa20/20 Core function as well as a set of reduced Salsa20/8 Core and Salsa20/12 Core functions

val salsa20_8_core : string -> string

salsa20_8_core input is output, the Salsa20/8 Core function. input must be 16 blocks of 32 bits.

  • raises Invalid_argument

    if input is not of the correct size
